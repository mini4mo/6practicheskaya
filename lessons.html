Уроки и курсы

Здесь вы найдете бесплатные онлайн-курсы, видеоуроки и практические задания, которые помогут вам освоить программирование. Начните с основ и постепенно переходите к более сложным темам. Практика — ключ к успеху!

  • Coursera - Курсы от ведущих университетов. Многие курсы бесплатны, а за сертификат можно заплатить.

Практические задания

Задание 1: Привет, мир!

Напишите программу, которая выводит "Привет, мир!" в консоль. Это классическое задание для начала знакомства с любым языком программирования.

# Python
print("Привет, мир!")

Задание 2: Простой калькулятор

Создайте программу, которая принимает два числа и выводит их сумму. Это задание поможет вам понять, как работать с вводом данных и базовыми операциями.

# Python
num1 = float(input("Введите первое число: "))
num2 = float(input("Введите второе число: "))
print("Сумма:", num1 + num2)

Задание 3: Проверка на четность

Напишите программу, которая проверяет, является ли введенное число четным или нечетным. Это задание познакомит вас с условными операторами.

# Python
num = int(input("Введите число: "))
if num % 2 == 0:
    print("Четное")
else:
    print("Нечетное")
